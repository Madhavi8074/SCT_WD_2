This project is developed as part of the SkillCraft Technology Web Development Internship.

## Description
An interactive stopwatch web application that allows users to start, pause, reset, and record lap times with accurate time tracking.

- Start, Pause, Reset functionality
- Lap time recording
- Accurate timer using JavaScript
- Responsive UI
